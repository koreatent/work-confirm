어닝 견적 계산기 v1

가로(mm), 돌출, 물받이 여부만 입력하면 즉시 견적 금액이 계산된다.
최종 견적 금액
788,000원
즉시 계산
가로 환산
4.5m
적용 단가
175,000원
기본금액787,500원
1000원 단위 올림788,000원
물받이0원
최종합계788,000원
계산 규칙: 가로(mm)를 m로 변환 → 돌출 단가 적용 → 기본금액 계산 → 기본금액만 1000원 단위 올림 → 물받이 옵션 추가.
단가표: 1.5m = 165,000원 / 2.0m = 175,000원 / 2.5m = 185,000원 / 3.0m = 210,000원
물받이: 3m당 40,000원, 개수는 올림 처리
